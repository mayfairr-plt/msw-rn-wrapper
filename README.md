# msw-rn-wrapper

Wrapper for MSW with React Native Support 

## Installation

```sh
npm install msw-rn-wrapper
```

## Usage

```js
import { multiply } from 'msw-rn-wrapper';

// ...

const result = await multiply(3, 7);
```
